# Project Summary
The project is an advanced crypto trading bot system designed for automated trading on platforms such as Bybit and Binance. It offers a web-based interface for users to configure trading strategies, manage accounts, and monitor performance in real-time. The backend is powered by FastAPI, ensuring efficient API services for seamless integration with the frontend.

# Project Module Description
- **Frontend**: A user-friendly interface for setting up trading parameters and tracking performance metrics.
- **Backend**: Comprehensive API services that manage configurations, account operations, and trading logic.
- **Bot Controller**: Central component that oversees trading operations and communicates with crypto exchanges.

# File Description Inventory
- **app_state.py**: Manages the global state for the BotController instance.
- **bot_fib_scoring.py**: Contains the logic for controlling the trading bot.
- **routes/**: Directory holding all API route definitions.
- **server.py**: Entry point for the application, initializes FastAPI and includes routes.
- **services/**: Contains business logic for different functionalities.
- **templates/**: HTML files used to render the frontend views.

# Technology Stack
- **FastAPI**: Backend framework for building APIs.
- **Jinja2**: Templating engine for rendering HTML.
- **WebSocket**: For real-time data updates.
- **Python**: Programming language for backend logic.
- **HTML/CSS**: For frontend design.
